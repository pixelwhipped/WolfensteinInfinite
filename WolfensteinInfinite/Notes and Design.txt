Suggested order of attack:
Start with Implementation in this order since they unblock everything else:

ResetGame — player death is currently broken which makes testing painful
Mission objectives — small, self contained, completes the core loop
Map toggle — quality of life for testing everything else
Level scaling curve — needed before any balance work is meaningful
Special levels every 10th — fun milestone, needs prefab map loader
Run arguments — enables mod testing workflow
Full map / non-composable sections — extends modding capability
LightBlur config option — trivial, do it alongside any other config work

Then Polish roughly:

Enemy animations — dying and firing are currently broken which makes combat feel wrong
Map generation improvements — once objectives and scaling are in you have real feedback
Code cleanup — do this in passes alongside features not as a big bang
Map editor optimization — WPF slowness on large section lists is likely a missing virtualization on the list control, probably a 30 minute fix

What I think is missing from your list:

Audio ambience / music state — menu music vs in-game music transition. You have audio plumbing but no state-driven music switching mentioned
Difficulty selection on new game feeds into enemy scaling but I don't see it verified as working end to end
Save file versioning — if you add fields to SaveGame or Config mid-development your existing saves will break silently. A Version int and migration path saves pain later
Credits screen — you mentioned ID Software love in the README but an in-game credits state costs almost nothing and feels right for a tribute game
Error / exception surface — right now if map generation throws the game likely hard crashes. A top-level exception handler that writes a log and shows a friendly message matters before any public release

On publishing — releasing with the Demo shareware files is exactly the right call legally and ethically. The shareware episode was explicitly released for free distribution. Just make the README crystal clear about where to get the full game files if people want them.
